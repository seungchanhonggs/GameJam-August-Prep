## 현재 확정된 v2 규칙

### 자유 회전 조작

- 좌우 탭 및 120도 단위 자리 교체를 폐기했다.
- 플레이어가 전장 원을 손가락으로 돌리면 세 영웅과 공격 범위가 휠처럼 연속 회전한다.
- 스냅 없이 작은 각도도 그대로 유지한다.
- 코어 중앙 탭은 회전 입력과 분리된 투명 버튼으로 처리한다.

### 영웅별 공격 범위

- 영웅이 120도 전체를 담당하지 않는다.
- 궁수: 38도, 사거리 340. 좁고 긴 단일 공격.
- 마도사: 82도, 사거리 255. 중거리 광역 공격.
- 수호자: 104도, 사거리 190. 넓고 짧은 방어.
- 적이 현재 영웅 방향의 부채꼴 각도와 사거리 안에 모두 들어와야 공격 대상이 된다.
- 수치는 밸런스 패널에서 수정 가능하다.

### 자동 필살기

- 영웅별 필살기 드래그 조작을 제거했다.
- 충전 완료 후 해당 영웅 범위 안에 적이 있으면 자동 발동한다.
- 궁수: 다중 화살 집중 발사.
- 마도사: 큰 광역 투사체.
- 수호자: 현재 바라보는 방향 앞에 방어벽 자동 생성.
- 하단 원형 게이지는 자동 필살기 충전을 표시한다.

### 적 생성과 이동

- 기존 3개 고정 레인과 레인 번호 의존을 제거했다.
- 일반 적은 원 둘레 0~360도의 무작위 각도에서 계속 생성된다.
- 군집 적은 한 진입각 주변의 접선 방향으로 퍼져 등장한다.
- 큰 웨이브는 웨이브마다 새로운 집중 진입각을 선택한다.
- 보스도 매 판 무작위 방향에서 등장한다.
- 적은 코어로 직진하며 영웅이나 벽이 실제 경로를 막으면 충돌한다.

## 구현된 코어 링크: 공명 회전

### 충전

- 코어 둘레에 궁수 노랑, 마도사 파랑, 수호자 초록 조각이 있다.
- 궁수와 마도사의 자동 필살기 성공 시 해당 조각이 34 충전된다.
- 수호자 벽이 처음 적을 막을 때 초록 조각이 34 충전된다.
- 생존 영웅의 조각이 모두 100이면 코어에 `LINK / READY`가 표시된다.

### 발동과 콤보

- 준비된 코어를 탭하면 6초간 공명 회전이 발동한다.
- 코어와 생존 영웅 사이에 밝은 연결선과 이동 입자가 표시된다.
- 링크 중 휠을 돌려 같은 적을 서로 다른 영웅 범위에 연속해서 넣는다.
- 적 하나는 링크 한 번 동안 영웅별 효과를 한 번씩만 받는다.
- 궁수: 24 피해.
- 마도사: 18 피해.
- 수호자: 8 피해, 바깥쪽 밀어내기 32, 짧은 감속.
- 모든 생존 영웅의 효과가 같은 적에게 연결되면 생존 영웅 1명당 30의 코어 번개가 발동한다.
- 링크 중 영웅이 죽으면 필요한 역할 수와 번개 피해가 즉시 생존 인원에 맞춰 줄어든다.

## 현재 가장 중요한 미해결 문제

사용자가 코어 링크 사용법을 직관적으로 이해하지 못했다. 규칙 설명은 가능하지만 게임 화면의 피드백이 부족하다.

다음 우선 작업:

1. 링크 중 적 머리 위에 노랑, 파랑, 초록 3개의 작은 연결 상태 표시를 추가한다.
2. 해당 영웅 범위에 적중하면 대응 색상이 켜지게 한다.
3. 모든 필요한 색이 켜질 때 코어 번개가 발동했다는 것을 강하게 보여준다.
4. 코어 완충 시 단순 `READY`보다 명확한 탭 유도 펄스와 상태 변화를 추가한다.
5. 텍스트 설명에 의존하지 않고 충전 → 탭 → 회전 연결의 흐름을 시각적으로 이해할 수 있는지 테스트한다.
